# 32-bit ALU Design (Verilog + ASIC Flow)

## 📌 Overview
This project implements a 32-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## ⚙️ Features
- Arithmetic: ADD, SUB
- Logical: AND, OR, XOR, XNOR, NOT
- Flags: Zero, Carry, Sign

## 🛠 Tools Used
- Verilog HDL
- Cadence Genus (Synthesis)
- Cadence Innovus (Physical Design)

## 📊 Results
- Power: ~88 µW
- Area: 338 cells
- Timing: Fully optimized

## 🚀 Applications
- CPUs
- DSP systems
- Embedded systems

## 📂 Structure
- `src/` → Design files
- `tb/` → Testbench
- `docs/` → Project report
